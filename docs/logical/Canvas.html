constructor Drawing

protected Drawing(String name, Number width, Number height, [String color])
Creates an instance of a Drawing and inserts a canvas tag onto the web page.
Parameters:
name An instance name for this drawing. This will used be the ID of the HTML canvas element.
width The width in pixels of the canvas.
height The height in pixels of the canvas.
[color] The background colour of the drawing. Either a CSS colour name or a hex value.
Returns:
Nothing
