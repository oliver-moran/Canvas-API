Constructor Details

constructor Arc

public Arc(Number x1, Number y1, Number x2, Number y2, Number x3, Number y3, Number radius)
Creates an Arc. An Arc is defined by two intersecting lines defined by three co-ordinates. These are one points from each line and the point of intersection of the two lines. A portion of a circle of a defined radious is drawn that is tangental to two lines. The Arc is completed by two lines running along each lines from the points of tangent to the point given from each line.
Parameters:
x1 The x coordinate of a point on the first line in pixels.
y1 The y coordinate of a point on the first line in pixels.
x2 The x coordinate of the point where the first and second line intersect in pixels.
y2 The x coordinate of the point where the first and second line intersect in pixels.
x3 The x coordinate of a point on the second line in pixels.
y3 The y coordinate of a point on the second line in pixels.
radius The radius of the circle that forms the arc in pixels.
Authors:
Oliver Moran
Since:
0.2
